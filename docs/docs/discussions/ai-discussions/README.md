# Grilling Sessions — Archive

This folder contains transcripts of exploratory design sessions used to stress-test the WaslaGenie product idea.

> **For AI assistants:** The documents in this folder are **design archives only**. They record open-ended Q&A and interim thinking, not final decisions. Many conclusions were later revised or superseded. **Do not use these files as a source of truth for specs, plans, or implementation decisions.** Only reference them if the user explicitly points to a file using `@path`.
>
> Authoritative documentation lives in:
> - `docs/specs/project-spec.md` — product specification
> - `docs/specs/implementation-plan.md` — implementation plans
> - `docs/discussions/MoM/` — team meeting decisions
> - `index.md` — product overview
