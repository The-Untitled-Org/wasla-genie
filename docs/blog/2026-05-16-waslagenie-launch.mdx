# Introducing WaslaGenie

**The universal synchronization layer for AI agent orchestrators is here.**

We've all been there. You build an agent in Claude Code. It doesn't exist in Gemini CLI. You create an MCP in one tool. It's gone in another. You copy-paste configs across five different orchestrators, hoping they stay in sync.

Not anymore.

{/* truncate */}

## The Problem We're Solving

Working across multiple AI orchestrators creates a painful workflow:

- **Duplication**: Agents, MCPs, and configurations exist in multiple places
- **Inconsistency**: Changes in one tool don't propagate to others
- **Maintenance overhead**: Every update requires manual sync across all tools
- **Lost productivity**: Time wasted copy-pasting instead of building

## What is WaslaGenie?

WaslaGenie is a daemon that scans all your AI orchestrators, discovers what you've built, and keeps everything in sync—without duplicating a single file.

### How It Works

1. **Install**: Register WaslaGenie as a native skill in each orchestrator
2. **Sync**: Scan all tools, discover agents and MCPs
3. **Connect**: Write minimal stubs that reference originals
4. **Watch**: Let the daemon auto-sync as you create and edit

Your agents live where they were born. WaslaGenie makes sure every tool can find them.

## Key Features

✅ **Multi-tool support** — Claude Code, Gemini CLI, OpenCode  
✅ **Zero duplication** — Originals stay where they were created  
✅ **Automatic sync** — Daemon watches and updates stubs on change  
✅ **Conflict resolution** — Latest is greatest strategy  
✅ **Zero friction** — Create where you're comfortable  

## The MVP Is Here

We're launching WaslaGenie MVP with support for:

- **Asset types**: Agents + MCPs
- **Tools**: Claude Code, Gemini CLI, OpenCode
- **Sync modes**: Manual command + daemon watching
- **Scope**: User-level and workspace-level registry

## Get Started

```bash
npx wasla-genie install      # First-time setup
waslagenie sync              # Scan and sync your assets
waslagenie status            # View everything
waslagenie watch             # Auto-sync mode
```
