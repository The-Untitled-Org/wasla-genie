# Roadmap

WaslaGenie is currently in **alpha**. Here's what's planned:

## MVP (Current)

- [x] Asset discovery across Claude Code, Gemini CLI, OpenCode
- [x] Stub-based synchronization
- [x] Conflict resolution (latest is greatest)
- [x] Manual sync command
- [x] Daemon mode with file watching
- [x] Registry persistence

## v1.0

- [ ] CLI polish and documentation
- [ ] Comprehensive test coverage
- [ ] Cross-platform testing (macOS, Linux, Windows)
- [ ] Error recovery and edge cases
- [ ] Performance optimization for large asset counts

## v1.1

- [ ] npm package publication
- [ ] GitHub CI/CD workflows
- [ ] Adapter for additional tools
- [ ] Web dashboard for sync status
- [ ] Backup and restore functionality

## Future

- [ ] Real-time sync via webhooks
- [ ] Selective sync (include/exclude patterns)
- [ ] Asset versioning and history
- [ ] Multi-user workspace sync
- [ ] Cloud-backed registry
- [ ] BYO (Bring Your Own) adapter system
